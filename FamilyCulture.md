# 家风、家训、家规、家魂

## 适用场景

- 想写家训，但怕写成空口号
- 想做家庭公约、家庭文化、公序良俗
- 家里有爱，但长期没有规则，反复内耗
- 想给孩子和下一代留下精神底座

## 核心口径

> 家风是气候，家训是地图，家规是流程，家魂是方向。

四者不能混：

- 家风：日常行为气候
- 家训：原则语言
- 家规：执行条目
- 家魂：长期方向

## 最稳的生成顺序

```text
先收真实故事
  -> 再提炼 3-5 条原则
  -> 再把原则落成家规
  -> 再试运行
  -> 再年度复盘升级
```

## 先收什么故事

优先收这些：

- 谁在关键时刻扛过事
- 谁守信过
- 谁因为失信吃过大亏
- 谁从低谷里爬起来
- 家里最珍贵的劳动、读书、照护、利他记忆

原则：不从空话开始，从这个家真实发生过的事开始。

## 家训提炼法

把故事压缩成短句，但不要写成口号墙。

示例方向：

- 说话算数
- 钱要明白
- 长辈要敬，边界要清
- 孩子先做人，再成事
- 先把小事做实，再谈大的梦想

## 家规落地法

家规至少覆盖这些维度：

| 维度 | 例子 |
|---|---|
| 作息 | 晚上几点停手机，几点睡 |
| 读书 | 每月一本书，家里定期交流 |
| 金钱 | 大额支出先商量，借钱留记录 |
| 沟通 | 不在情绪顶峰决策，不让孩子传话 |
| 照护 | 老人支持节奏和责任人 |
| 节日 | 年会、祭祖、生日、感恩环节 |
| 会议 | 每月一次家庭复盘 |

## 家庭公约 1.0

最稳的做法不是一上来做百年家训，而是先做一版可以试运行 30 天的家庭公约。

### 建议结构

1. 我们家最重视什么。
2. 我们家绝不能碰什么。
3. 日常最少执行的三条规则。
4. 出现分歧时怎么开口。
5. 什么时候复盘。

## 绝对不要

- 直接抄曾国藩家书、颜氏家训、朱子家训当成自己家的东西
- 只有标语，没有动作
- 只有控制条款，没有大人示范
- 只要求孩子，不要求父母和夫妻

## 回答时的稳定落点

- 先帮用户收故事
- 再提原则
- 再给公约 1.0
- 再给 30 天试运行和复盘方法
