# 夫妻与小家庭同盟

## 适用场景

- 夫妻吵架、冷战、反复翻旧账
- 沟通失序、分工失衡、钱的问题
- 老人介入夫妻关系
- 孩子被夹在夫妻中间
- 关系里出现控制、羞辱、威胁或安全风险

## 核心口径

> 夫妻关系不是审判场，而是小家庭的同盟场。

很多婚姻不是没有感情，是一开口就进了法庭模式：只剩证据、输赢、指控，没有事实、责任、修复和同盟。

## 先判断属于哪一类

1. 沟通问题：话越说越伤，表达失真。
2. 分工问题：家务、孩子、钱、老人、事业失衡。
3. 外部介入：父母越界、亲属站队、第三方压力。
4. 安全问题：暴力、威胁、跟踪、成瘾、严重失控。

## 默认处理顺序

```text
先停法庭模式
  -> 退回共同事实
  -> 明确小家庭优先级
  -> 再谈分工、边界和协同
  -> 最后决定修复、暂停还是止损
```

## 夫妻对齐的三个关键点

### 1. 重大事项先内部商量

包括：

- 孩子教育
- 大额花费
- 老人照护安排
- 节日节奏
- 对外统一口径

### 2. 不让孩子承担婚姻重量

孩子不能：

- 传话
- 站队
- 安慰父母
- 承接父母的怨气

### 3. 老人介入要守边界

长辈可以被尊重、被照顾、被沟通，但不能替夫妻做最终决定。

## 可直接使用的动作

### 15 分钟夫妻会谈

每次只谈一个议题：

1. 这件事的共同事实是什么。
2. 我真正担心的是什么。
3. 我希望你承担的具体动作是什么。
4. 我愿意承担的动作是什么。
5. 什么时候复盘。

### 夫妻边界句式

- 我想先和你站在一边，再一起看问题。
- 这件事先不在老人面前讨论，我们回去自己定。
- 你现在的感受我听到了，我们先把事实说完整。
- 我们先不翻旧账，只处理这一次该怎么改。

## 安全红线

以下情况不能再包装成“夫妻功课”：

- 身体暴力
- 砸东西、限制自由、经济控制
- 跟踪、恐吓、拿孩子威胁
- 严重成瘾和高危失控行为

这时的顺序只能是：

```text
先离开危险
  -> 联系可信任的人
  -> 保留记录
  -> 用当地紧急服务、警方、社工、法律或反家暴资源保护自己
```

## 回答时的稳定落点

- 先把夫妻从“对抗”带回“同盟”
- 再把情绪带回事实和分工
- 最后给一场会谈或一个边界动作
