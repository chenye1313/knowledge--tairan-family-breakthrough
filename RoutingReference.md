# 家庭破局知识路由参考

## 固定底座

任何家庭问题，默认先读：

1. `TairanVoice.md`
2. `SystemBoundary.md`
3. `FamilyOverview.md`

这样做的目的，是先锁定语气、边界和家庭结构，不让回答漂成泛泛的人生建议。

## 再按问题域补读

### 父母、尽孝、养老、陪诊、父母去世

读：`Parents.md`

### 夫妻、婚姻、伴侣、冷战、老人介入

读：`Marriage.md`

### 孩子、亲子、教育、青春期、学习焦虑

读：`Parenting.md`

### 兄弟姐妹、赡养、分工、费用、协同

读：`Siblings.md`

### 家风、家训、家规、家魂、家庭公约

读：`FamilyCulture.md`

### 家族理事会、家庭会议、年会、祭祖、家族基金

读：`FamilyGovernance.md`

### 家业、家产、继承、股权、接班、家国文化

读：`FamilyLegacy.md`

### 功课、抽离、填空、感恩、重塑

读：`PracticeMethods.md`

### 模板、清单、议程、表格

读：`PracticeTools.md`

### 术语定义和概念澄清

读：`KeyTerms.md`

### 出处、原话、能否正式发布、证据层级

读：`SourceEvidence.md`

## 组合路由建议

### 家暴或高风险关系

`TairanVoice` + `SystemBoundary` + `Marriage` + 必要时 `SourceEvidence`

### 兄弟姐妹赡养老人

`TairanVoice` + `SystemBoundary` + `Parents` + `Siblings` + `PracticeTools`

### 写家庭公约

`TairanVoice` + `SystemBoundary` + `FamilyCulture` + `PracticeTools`

### 家族理事会起步

`TairanVoice` + `SystemBoundary` + `FamilyGovernance` + `PracticeTools`

### 家业传承与接班

`TairanVoice` + `SystemBoundary` + `FamilyLegacy` + 必要时 `SourceEvidence`

## 不要这么路由

- 不要一上来一次性全读所有文件。
- 不要遇到家庭问题先读环境和号码。
- 不要只读方法论，不读家庭结构文件。
- 不要只读传统经典背景，不读 3.0 主库口径。
