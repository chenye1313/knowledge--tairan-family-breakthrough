# 家庭场景下的功课方法

## 方法底线

第二卷的方法在家庭场景里依然成立，但顺序不能错：

```text
先现实边界
  -> 再情绪稳定
  -> 再做功课
  -> 再回到行动
```

功课不是：

- 忍耐
- 自责
- 替别人洗白
- 不处理现实问题

## 家庭场景的功课主线

### 1. 明心见性

看见我在这件家庭事件里：

- 害怕什么
- 抓着什么定义
- 在重复什么旧反应
- 把谁放错位置了

### 2. 抽离

从“我又被针对了”“他们永远不会变”这些剧情里退半步，先看事实。

### 3. 填空

不是骂自己，而是看自己内在缺的是什么：

- 安全感
- 被看见
- 认可
- 边界
- 被支持的感受

### 4. 感恩

不是把坏事说成好事，而是看见：

- 我真正学到了什么
- 我不想再重复什么
- 我该把主导权拿回来在哪一处

### 5. 重塑

把新的位置、表达和动作写回现实。

## 神游八荒与逍遥有为的家庭版

- 神游八荒：不是脑补一家人立刻圆满，而是先打破“只能继续这样”的旧剧本。
- 逍遥有为：不是躲开家庭责任，而是在不被旧模式绑死的情况下，做有效创造。

## 家庭场景里的典型应用

### 父母议题

先看我是在尽责，还是在用讨好换认可。

### 夫妻议题

先看我是在解决事实，还是在争输赢和清白。

### 亲子议题

先看我是为了孩子成长，还是在借孩子安放自己的焦虑。

### 兄弟姐妹议题

先看我是在做协同，还是在用委屈硬扛后等别人愧疚。

## 四个提醒

1. 功课不能取消报警、就医、法律和社工支持。
2. 感恩不能覆盖伤害事实。
3. 向内求不是替他人免责。
4. 做完功课一定要回行动，否则只是明白了一次。
