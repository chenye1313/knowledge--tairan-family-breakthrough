# 家族理事会、年会与祭祖

## 适用场景

- 家族人很多，越来越散
- 想做家族理事会，但怕变成权力中心
- 想开家庭会议、家族年会、祭祖仪式
- 想处理老人、孩子、节日、基金、记忆整理等公共事务

## 核心口径

> 家族理事会是协同接口，不是裁判权。

> 家族年会不是比较场，是让来处、长辈、孩子和传承变得可见。

> 祭祖重点在看见来处、摆正位置和激活感恩，不在迷信决定论。

## 家族理事会的边界

### 可以处理

- 老人照护
- 家族年会
- 祭祖安排
- 教育奖励
- 家族基金
- 家族故事整理
- 公共规则和节奏

### 不能处理

- 审判夫妻私事
- 夺取小家庭自主权
- 替代法律合同
- 用长辈身份压结论

## 理事会的最小起步法

第一次不要做大，先从一个小议题开始：

- 春节年会怎么开
- 老人生日怎么安排
- 谁来整理家族故事
- 教育奖励怎么做

## 会议角色

- 主持人：控节奏，不带头审判
- 记录人：记结论、负责人、时间点
- 时间管理员：防止无限拉扯
- 议题提出人：讲清事实和目标
- 执行负责人：会后推进

## 一场会议的顺序

```text
先讲事实
  -> 再讲需求
  -> 再讲可承接资源
  -> 再定决定
  -> 再定负责人和复盘时间
```

## 年会和祭祖的稳妥结构

### 家族年会

1. 先敬长辈、确认来处。
2. 回看一年谁做了哪些实事。
3. 表彰具体贡献，不搞攀比炫耀。
4. 让孩子听见家族故事，而不是只被盘问成绩收入。
5. 定下一年的一个公共动作。

### 祭祖

1. 庄重、简洁、可执行。
2. 重点在感恩、记忆、位置和教育。
3. 不神化，不恐吓，不做决定论。
4. 给孩子讲“我们从哪里来”。

## 回答时可直接给的动作

- 一次 45 分钟家庭会议议程
- 第一次理事会角色表
- 一页决策记录模板
- 一版年会流程
- 一版祭祖注意事项

## 风险边界

如果家族内部已经涉及：

- 继承和产权争议
- 长期侵占和失信
- 人身安全
- 严重情绪对抗

那理事会只能做沟通辅助，不能替代法律和专业处置。
