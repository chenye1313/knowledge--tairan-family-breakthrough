# 父母与根

## 适用场景

- 父母控制、指责、抱怨“不孝”
- 养老、陪诊、住院、失能照护
- 想修复和父母的关系，但不知道怎么开始
- 父母老去、父母去世、遗憾与哀伤整理
- 想尽孝，但不想再被道德压垮

## 核心口径

> 父母就是风水，讲的是根、托底、位置和家庭风向，不是让你把自己交出去。

这里有三层意思：

1. 看见根：承认父母在生命里的来处和托底。
2. 看见责任：能承担的，就把它变成现实支持。
3. 看见边界：有伤害、有控制、有侵占时，安全边界不能取消。

## 分析顺序

### 1. 先看父母真实处境

- 身体和医疗
- 情绪状态
- 经济状况
- 居住安排
- 陪伴频率
- 应急联系人

### 2. 再看子女真实承接能力

- 时间
- 金钱
- 陪诊
- 日常联系
- 节日陪伴
- 异地支持

### 3. 再看关系里卡住的是什么

- 旧伤未整理
- 期待不说清
- 边界长期被穿透
- 兄弟姐妹分工失衡
- 父母把情绪和控制包装成“爱”

| 费用安排 |  |  |  |  |
| 节日陪伴 |  |  |  |  |

### 边界句式

- 我会承担我能承担的部分，但我不会接受辱骂和威胁。
- 这件事我愿意做，我们把时间和方式说清楚。
- 你现在情绪很重，我们先停一下，等平稳一点再谈具体安排。
- 我尊重你是父母，但这件事需要我们用事实和安排来谈。

## 父母离世与哀伤整理

遇到父母去世、临终、重大病痛，不要一上来讲大道理。顺序应该是：

```text
先处理现实事务
  -> 允许哀伤
  -> 整理未完成的话
  -> 形成纪念与传承动作
  -> 再慢慢回到生命理解
```

可以给的动作：

- 写一封“还没来得及说”的信
- 和兄弟姐妹一起整理父母一生重要故事
- 做一个纪念日清单
- 把照护、祭扫、纪念的节奏定下来

## 绝对边界

以下情况不能被包装成“尽孝”：

- 家暴或严重精神控制
- 财务侵占
- 逼迫放弃婚姻和小家庭基本边界
- 长期羞辱、威胁、跟踪
- 用疾病或离世恐吓操控

这时要先保护自己和小家庭，再在安全距离里完成理解和整理。

## 回答时的稳定落点

- 既保留“父母就是风水”的根和责任
- 又守住事实、边界和现实照护
- 尽量给一次沟通、一张支持表或一周节奏
